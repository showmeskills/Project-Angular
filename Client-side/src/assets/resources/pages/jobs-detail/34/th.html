หน้าที่ความรับผิดชอบ:

1. รับผิดชอบในการออกแบบสถาปัตยกรรมและการพัฒนาระบบส่งเสริมสิทธิพิเศษ ระบบการเติบโตของผู้ใช้ และระบบแนะนำเพื่อน

2. มีส่วนร่วมในการเพิ่มประสิทธิภาพ การรับประกันความเสถียร การแก้ไขปัญหา และการแก้ปัญหา





ข้อกำหนดของงาน:

1. วุฒิปริญญาตรีขึ้นไป สาขาที่เกี่ยวข้องกับคอมพิวเตอร์ มีความรู้ด้านคอมพิวเตอร์และการทำงานเป็นทีมได้ดี

2. โครงสร้างข้อมูลที่มั่นคงและรากฐานอัลกอริธึม ความสามารถในการใช้งานทางวิศวกรรมที่ยอดเยี่ยม มีประสบการณ์ในการพัฒนา Java มากกว่า 3 ปี เชี่ยวชาญในการเขียนโปรแกรมแบบมัลติเธรด และประสบการณ์ในการปรับแต่งประสิทธิภาพ

3. มีประสบการณ์ในการพัฒนาบริการออนไลน์แบบกระจายที่มีความพร้อมใช้งานสูง มีความเชี่ยวชาญในหลักการทางเทคนิคและรายละเอียด เช่น การกระจาย การแคช คิวข้อความ ฯลฯ

4. ความเข้าใจทางธุรกิจที่ยอดเยี่ยมและทักษะในการวิเคราะห์ปัญหาและการแก้ปัญหา การริเริ่มที่แข็งแกร่งและแรงจูงใจในตนเอง เต็มไปด้วยความกระตือรือร้นสำหรับเทคโนโลยีที่ทันสมัยและความท้าทายทางธุรกิจ

5. ความสามารถในการคิดเชิงตรรกะที่ดี วิเคราะห์ ชักนำ และแก้ปัญหาได้ดี

6. เข้าร่วมในการวิเคราะห์ระบบ การออกแบบทั่วไป และการออกแบบซอฟต์แวร์ธุรกิจโดยละเอียด

7. รับผิดชอบในการใช้งานการเข้ารหัสซอฟต์แวร์ การทดสอบหน่วย การเขียนและการบำรุงรักษาเอกสารที่เกี่ยวข้องในกระบวนการพัฒนาระบบ





ต้องการสิ่งต่อไปนี้:

1. มีประสบการณ์ในการพัฒนาระบบการทำงานพร้อมกันสูง

2. มีประสบการณ์ในการออกแบบระบบแบบกระจายขนาดใหญ่

3. มีประสบการณ์ในการออกแบบและพัฒนาการแลกเปลี่ยนสกุลเงินดิจิตอล

4. รักอีสปอร์ตและกีฬา
