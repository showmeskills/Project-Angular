หน้าที่ความรับผิดชอบ:

1. รับผิดชอบการแก้ไขปัญหาฉุกเฉินและการตรวจสอบสาเหตุหลังเหตุการณ์ปัญหาออนไลน์

2. ปัญหาที่ไม่สามารถจัดการได้ในทันที ควรรายงานไปยังบุคลากรที่เกี่ยวข้อง และควรติดตามความคืบหน้าในการดำเนินการของปัญหา

3. บันทึกปัญหาออนไลน์และอัปเดตไฟล์การจัดการปัญหา

4. ตรวจสอบบริการออนไลน์ทั้งหมดเพื่อให้แน่ใจว่าบริการทำงานตามปกติ

5. ทำงานร่วมกันและสื่อสารอย่างใกล้ชิดกับนักพัฒนาซอฟต์แวร์

6. ช่วยในการพัฒนาโปรแกรมอัตโนมัติและตรวจสอบว่าฟังก์ชันออนไลน์ผิดปกติหรือไม่





เงื่อนไข:

A. เคยใช้ AWS หรือแพลตฟอร์มบริการคลาวด์อื่นๆ (GCP/Azure)

B. ทำความเข้าใจสถาปัตยกรรมบริการเครือข่าย (Networking/DNS/Load Balancing…)

C. ใช้ระบบปฏิบัติการที่เกี่ยวข้องกับ Linux

D. คุ้นเคยกับเซิร์ฟเวอร์ Windows

ประสบการณ์การพัฒนาซอฟต์แวร์ E. C#





เงื่อนไขโบนัส:

1. ความสามารถในการสื่อสารภาษาอังกฤษของ Jichu

2. มีใจรักในการบริการลูกค้า

3. การต้านทานแรงกดสูง/ความสามารถในการรับมือวิกฤตที่ดี

4. ความสามารถในการเรียนรู้ที่แข็งแกร่งและเต็มใจที่จะซึมซับความรู้ทางวิชาชีพใหม่ ๆ

5. ต้องสามารถทำงานกะ/การจัดตารางเวลาได้
