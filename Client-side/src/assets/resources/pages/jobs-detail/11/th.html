ข้อกำหนดของงาน:

รับผิดชอบการวางแผน UX และการออกแบบ UI ของ Web / Mobile / APP ปรับปรุงการออกแบบส่วนต่อประสานการทำงานของผลิตภัณฑ์และเสริมสร้างการออกแบบกิจกรรมการตลาดผลิตภัณฑ์ด้วยความสามารถในการแสดงออกที่ดีและความสามารถในการคิดเชิงตรรกะ ด้วยการใช้ความคิดสร้างสรรค์ การจัดองค์ประกอบภาพและการออกแบบที่สวยงามของเลย์เอาต์ และความละเอียดอ่อนในการผลิตรายละเอียดของภาพ มันถูกนำไปใช้กับผลิตภัณฑ์ของบริษัท และร่วมมือกับโครงการและการพัฒนาเพื่อดำเนินธุรกิจต่างๆ ของบริษัท





เงื่อนไขของสกิล:

1. คุ้นเคยกับส่วนต่อประสานการพัฒนาผู้ใช้ของ Windows/Web/iOS/Android Base

2. ผู้สนับสนุนความต้องการของผู้ใช้ ผ่านกระบวนการคิดเชิงออกแบบ เปลี่ยนเป็นการออกแบบอินเทอร์เฟซและประสบการณ์การโต้ตอบกับผู้ใช้

3. คุ้นเคยกับซอฟต์แวร์ที่เกี่ยวข้องกับการออกแบบ เช่น Adobe Xd/photoshop/illustrator/zeplin เป็นต้น และผลิตเอกสารต้นแบบและข้อกำหนดตามข้อกำหนดของโครงการ

4. ด้วยความสามารถในการออกแบบภาพ ความใส่ใจในรายละเอียดและความคิดสร้างสรรค์

5. มีประสบการณ์ที่เกี่ยวข้องมากกว่า 2 ปี โปรดนำเอกสารที่เกี่ยวข้องกับประวัติย่อและผลงานมาสัมภาษณ์
