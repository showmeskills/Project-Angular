ข้อกำหนดของงาน:

1 เชี่ยวชาญการพัฒนาแอปข้ามแพลตฟอร์มใน Flutter และสามารถพัฒนา iOS หรือ Android ให้เสร็จสมบูรณ์ได้โดยอิสระ

2. คุ้นเคยกับการพัฒนาเนทีฟของ iOS หรือ Android และมีพื้นฐาน JAVA หรือ OC/Swift ที่มั่นคง

3. เชี่ยวชาญใน iOS, Android SDK และการเขียนโปรแกรมเครือข่าย, การเขียนโปรแกรมแบบมัลติเธรด, การพัฒนา UI

4. ทำความคุ้นเคยกับโครงสร้างอัลกอริธึมทั่วไปและรูปแบบการออกแบบ

5. มีส่วนร่วมในการกำหนดโซลูชันทางเทคนิคของลูกค้า และดำเนินการออกแบบหรือพัฒนาที่เกี่ยวข้อง





เงื่อนไขเพิ่มเติม:

ประสบการณ์การพัฒนาเทอร์มินัลมือถือ 1, 3 ปี พร้อมประสบการณ์การพัฒนาโครงการหลายโครงการ

2. มีประสบการณ์การพัฒนาระบบ IOS หรือ Android อย่างน้อย 2 ปี

3. นิสัยการเขียนโปรแกรมที่ดี ข้อกำหนดของโค้ด สามารถเขียนโค้ดได้เสถียร มีประสิทธิภาพ ง่ายต่อการบำรุงรักษา และขยายโค้ดได้

4. มีความสามารถที่แข็งแกร่งในการทนต่อแรงกดดัน การรับรู้ในทีมที่ดีและมีความรับผิดชอบสูง มีความสามารถในการสื่อสารและร่วมมือกับผู้คนได้ดี

5. ความสามารถทางเทคนิคที่แข็งแกร่ง

6. ผู้ที่มีความสามารถเป็นเลิศสามารถเป็นพันธมิตรด้านเทคนิคและเอาชนะความยากลำบากได้
