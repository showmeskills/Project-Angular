หน้าที่ความรับผิดชอบ:

1. รับผิดชอบในการสร้างเว็บไซต์ส่วนหน้า, การบำรุงรักษาตามปกติของการพัฒนา, การแก้ปัญหาออนไลน์แบบ on-call

2. ทำความคุ้นเคยกับเบราว์เซอร์ทั่วไป สามารถแก้ปัญหาความเข้ากันได้ของเบราว์เซอร์และการพัฒนาประสิทธิภาพได้อย่างมีประสิทธิภาพ

3. มีความตระหนักรู้ในทีมและสามารถทำงานร่วมกับสมาชิกในทีมคนอื่นๆ เพื่อพัฒนาข้อกำหนดและสนับสนุนการจัดการปัญหาฉุกเฉินทางออนไลน์ได้

4. นิสัยการพัฒนาโค้ดโปรแกรมที่ดี ให้ความสนใจกับ Clean code และหลักการออกแบบ OO และสามารถให้ความสนใจและเพิ่มประสิทธิภาพโค้ดของโปรแกรมต่อไปได้

5. เขียนการทดสอบหน่วย ผสานโค้ดโปรแกรมทดสอบ และผสานรวมกับทีมอย่างต่อเนื่องเพื่อนำเสนอโปรแกรมคุณภาพสูง

6. เข้าร่วมในการออกแบบและพัฒนาสถาปัตยกรรม และสนับสนุนการวิวัฒนาการและการปรับปรุงระบบอย่างต่อเนื่องเพื่อรับมือกับความต้องการที่เปลี่ยนแปลงอย่างรวดเร็ว





เงื่อนไข:

1. คุ้นเคยกับ Angular และมีประสบการณ์ในการพัฒนาเว็บแอปพลิเคชัน ES6/7 และการปรับแต่งประสิทธิภาพที่เกี่ยวข้อง

2. คุ้นเคยกับ RWD หรือ PWA และ UI Framework ทั่วไป (เช่น Bootstrap หรือ Framework อื่นที่คล้ายคลึงกัน)

3. มีความชำนาญในการใช้คุณลักษณะ CSS3, ข้อกำหนด HTML5 ล่าสุด และสามารถใช้คุณลักษณะ HTML5 เพื่อสร้าง WebApps ที่เข้ากันได้กับการเรียกดูอุปกรณ์เคลื่อนที่

4. ทำความคุ้นเคยกับการทดสอบหน่วย เทคนิคการจัดโครงสร้างใหม่และกรอบงานและเครื่องมือสำหรับการทดสอบอัตโนมัติ

5. ทำความคุ้นเคยกับเครื่องมือควบคุมเวอร์ชัน Git และประสบการณ์การพัฒนาร่วมกันหลายคน

6. ทำความคุ้นเคยกับการพัฒนาคอนเทนเนอร์ของ Docker





เงื่อนไขโบนัส:

1. ทำความคุ้นเคยกับ Jenkins หรือเครื่องมืออื่นๆ ที่เกี่ยวข้องกับ CI

2. ทำความคุ้นเคยกับ Selenium หรือเครื่องมือทดสอบอัตโนมัติอื่นๆ ที่เป็นประเภทเดียวกัน

3. คุ้นเคยกับ NodeJS/Python หรือมีประสบการณ์ในการพัฒนาฝั่งเซิร์ฟเวอร์

4. ทำความคุ้นเคยกับ GraphQL

5. มีแนวทางการพัฒนา TDD/BDD

6. มีประสบการณ์ในการใช้บริการระบบคลาวด์ GCP, AWS และ Azure

7. คุ้นเคยกับ UI Design หรือ User Experience และมีประสบการณ์การวางแผนการออกแบบที่เกี่ยวข้อง





เก่งเรื่องเครื่องมือ:

Python, AWS, HTML, CSS3, Angular
