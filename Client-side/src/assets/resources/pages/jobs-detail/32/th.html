เนื้อหางาน:

ความร่วมมือกับการดำเนินการไวยากรณ์ฐานข้อมูลทีมพัฒนา & การเขียนคำสั่ง T-SQL (SP)

ประสานงานกับทีมพัฒนาเพื่อจัดการกับปัญหาแบบเรียลไทม์ในการสืบค้นฐานข้อมูลและการยืนยันเชิงตรรกะ

ปรับไวยากรณ์ฐานข้อมูลเพื่อทำงานกับระบบที่มีอยู่ของทีมพัฒนา

งานบำรุงรักษาฐานข้อมูลเป็นระยะ (สำรอง/ตรวจสอบ/งานประจำ)

การปรับไวยากรณ์ของฐานข้อมูล (แผนปฏิบัติการ/การใช้ทรัพยากร)





เงื่อนไขการสมัคร:

คุ้นเคยกับฐานข้อมูลพื้นฐาน SQL syntax application (DDL/DML/DCL) คุ้นเคยกับ T-SQL เป็นอย่างดี

ประสบการณ์ฐานข้อมูลใดๆ (SQL Server, MySQL, Postgres ฯลฯ...) และความคุ้นเคยกับ SQL Server จะได้รับการพิจารณาเป็นพิเศษ





เงื่อนไขโบนัส:

ประสบการณ์การบริการ AWS หรือการรับรองที่เกี่ยวข้อง

ประสบการณ์ NOSQL หรือการรับรองที่เกี่ยวข้อง

ประสบการณ์การพัฒนาภาษาการเขียนโปรแกรมใดๆ





เงื่อนไขอื่นๆ:

เปิดใจในการสื่อสารและใช้ความระมัดระวัง

ฉันชอบการทำงานเป็นทีมมากกว่าความเป็นปัจเจก

มีความหลงใหลในการเรียนรู้เทคโนโลยีใหม่ๆ และความเต็มใจที่จะแบ่งปันกรอบความคิดแบบเติบโต
